# Pakistan Real Estate Market Analysis & Price Prediction

## Overview
This project presents a complete end-to-end data science pipeline using 168,000+ real estate property listings scraped from Zameen.com. The objective is to analyze market trends, extract business insights, and build machine learning models to predict property prices.

## Dataset
- 168,446 property records
- 20 original features
- Includes location, price, area, bedrooms, bathrooms, purpose, and geographic coordinates

## Project Pipeline
1. Data Cleaning & Preprocessing  
2. Exploratory Data Analysis (EDA)  
3. Feature Engineering  
4. Machine Learning Modeling  
5. Business Insight Generation  

## Key Insights
- Area size is the strongest factor affecting property prices.
- Location and city significantly influence property valuation.
- Bedrooms and bathrooms play an important role in pricing trends.
- Luxury properties cluster strongly in major metropolitan areas.
